ShaYa - 2023.07.03

发表于 2023-07-03 分类于 Diary

终于尝试背了日语，希望能够像写日记一样一直坚持下去。

标题是今天随机到的一首中 V 歌。。。的拼音，原标题就任君想象好了 www。

不是代指同 id 的某人。

说起来为什么突然像着了魔一样开始写日记呢？可能是因为最近入了《主播女孩重度依赖》的坑，然后看到作者 にゃるら 竟然！每天都会更新博客。每天啊！于是想要自己也试着去模仿一番。

正好自己有一个搁置很久没有使用的个人博客。想着每天做些什么的习惯也会让自己生活变得健康一点，于是就这么做了。最后还考虑到咱每天做了什么，还是简单做个总结比较好。这样容易看到哪些天比较摸鱼，哪些天比较卷。

这么说来其实思虑很深啊 www。不过能坚持几天呢？我也不知道。就拭目以待吧。

今天主要做了 Atiyah 第七章的一些习题，回忆了一些技巧。但回忆的还不够多！其实睡的比较多。昨天还是没怎么睡醒，今天继续补了会儿觉。

最重要的是，我今天，终于，开始背日语了！但是背了多少我也不知道。平假名我还是比较眼熟，但是某些片假名就比较难受，主要是常用词汇也很难看到片假名。

是的，我连五十音都没背完。千万别跟别人说啊，不然别人就知道我和他们的起点都不一样了 QaQ。

今天读到了 xYix 的书单，本来想仿照着做个自己的书单，结果发现自己没读完几本书。

当然，文学系的书籍本来就没怎么读。专业上的书籍则大多读完前面一半甚至不足一半就弃了。怎么能满足于一遍又一遍的基础呢？是该好好反省了。

我怎么还欠某只兔子一篇小短文。
